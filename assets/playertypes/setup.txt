How do you create a new playertype?

First, create a unique name for this playertype
Then add stats:
- speed
- health
- strength
- etc etc - tbd

Then add animation images
filepath will be relative to main.py
"assets/animations/"

Example:

{
    "name": "basic",
    "stats":{
        "health":100,
        "speed":100,
        "strength": 10
    },
    "animation":{
        # filepath will be "assets/animations/{ani_name}/{frame}"
        "idle": {
            "frames":["img_1.png"],
            "frame_time": [0.1]
        }
    }
}
